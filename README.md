# MOVIES-REACT
## Описание
Cервис Movies: Сервис, в котором можно найти фильмы по запросу и сохранить  
в личном кабинете.

Верстка адаптивного сайта по макету в Figma  

Макет: Dark-4

## Использованные технологии
1) HTML
2) CSS (flexbox, grid)
3) JavaScript, React.js, React-router-dom
